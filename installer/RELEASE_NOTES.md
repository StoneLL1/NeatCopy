# NeatCopy v1.0.0

NeatCopy 是一款 Windows 剪贴板文本清洗工具，常驻系统托盘，通过全局快捷键一键清洗文本格式，支持规则引擎与大模型两种模式。

## 安装说明

1. 下载 `NeatCopy_Setup_v1.0.0.exe`
2. 双击运行安装向导，按提示完成安装（需管理员权限）
3. 安装完成后可选择立即启动
4. 选中任意文字后按 `Ctrl+Shift+C` 触发清洗，直接粘贴即可

## 主要功能

- **规则清洗模式**：合并 PDF/CAJ 软换行、智能标点、中英文间距（Pangu 风格）、保护代码块和列表，离线可用
- **大模型模式**：接入任意 OpenAI 兼容接口，自定义 Prompt 实现翻译、摘要、润色等
- **全局热键**：`Ctrl+Shift+C`（默认），可选双击 `Ctrl+C`
- **系统托盘**：图标四态变色（空闲 / 处理中 / 成功 / 错误）
- **开机自启**：可在设置中启用

## 系统要求

- Windows 10 / 11（x64）
- 首次运行需要管理员权限

## 注意

- 大模型模式需在设置中自行填入 API Key
- 配置文件存储于 `%APPDATA%\NeatCopy\config.json`，仅本地保存，不上传
