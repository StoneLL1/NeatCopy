# NeatCopy Notion 风格 UI 重构设计文档

## 概述

将 NeatCopy 设置窗口从传统 Tab + GroupBox 布局重构为 Notion 风格的侧边栏导航布局，包含布局重构、视觉风格重设计、交互体验提升。

## 目标

- 布局结构改造：Tab 改为侧边栏导航
- 视觉风格重做：Notion 风格配色和间距
- 窗口可调整大小

## 约束

- 保持 PyQt6 技术栈
- 保持所有现有功能不变
- 动画效果最小化（用户要求）
- 在 worktree 中开发，不污染主分支

## 主内容区设计

### 视觉规则

```
┌─────────────────────────────────────────┐
│                                         │
│  页面标题（18px，加粗，主要色）          │
│                                         │
│  ── 标题文字 ──（12px，灰色）           │
│                                         │
│  ☐ 设置项 1                             │
│  ☐ 设置项 2                             │
│  热键：  [ctrl+shift+c]                 │
│                                         │
│  ─────────────────（1px 细线分隔）      │
│                                         │
│  ── 另一个标题 ──                       │
│                                         │
│  ☐ 设置项 3                             │
│                                         │
└─────────────────────────────────────────┘
```

### 核心改变

1. **去掉所有 GroupBox 边框和背景**
2. **区块标题**：`── 标题 ──` 样式的 QLabel（12px 灰色文字）
3. **区块分隔**：1px 水平线 QFrame(HLine)
4. **设置项间距**：8-10px（更多留白）
5. **滚动支持**：内容区有 QScrollArea
6. **右侧 padding**：24px

## 技术实现要点

### 新增组件

1. `SidebarWidget` — 自定义侧边栏组件
   - 继承 QWidget
   - 包含应用名称 Label
   - 导航按钮列表（QListWidget 或自定义按钮）
   - 选中状态切换信号

2. `ContentPage` — 内容页基类
   - 继承 QWidget
   - 包含页面标题
   - 滚动区域支持

### 修改组件

1. `SettingsWindow` — 主窗口
   - 移除 QTabWidget
   - 改用 QHBoxLayout：左侧 SidebarWidget + 右侧 QStackedWidget
   - 窗口大小可调整
   - resizeEvent 保存窗口尺寸

2. `src/ui/styles.py` — 样式模块
   - 新增侧边栏样式
   - 新增内容区样式
   - 新增区块标题样式

### 配置新增

```python
'ui': {
    'theme': 'light',
    'window_width': 700,
    'window_height': 550,
}
```

## 风险与缓解

| 风险 | 缓解措施 |
|------|----------|
| 大量代码重构可能引入 bug | 在 worktree 开发，保持测试通过 |
| 现有信号/槽连接可能断开 | 保留所有方法签名，只改 UI 结构 |
| 窗口大小持久化可能失败 | 使用已有的 config.set 机制 |
